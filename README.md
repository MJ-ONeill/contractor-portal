A dashboard for contractors to add their project on
The user can add the job name, then click to open the job portal
the user can then add all their expenses.
Eg. Store name where materials bought with dates, total paid
wages paid and specific dates paid to which worker
document their sub conteractor payments
and whatever else regarding that job

The total expenses is diplayed on the job card and on the top of the job portal
the user can mark a job as completed along with the start and finish date.

A nice orgeniser for contrators who don't want to have to many files saved or carry papers around the whole time

It's fully responsive on all screens.

Currently only saves too local storage but it has space for cross-device sync capabilities
